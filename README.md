# MongoDB Practice

This repository contains all the MongoDB commands I learned and practiced using the VS Code MongoDB Playground.

## What is inside?

- Database commands  
- Collection commands  
- Insert (one & many)  
- Read / Find commands  
- Update commands  
- Delete commands  

Everything is tested inside `.mongodb.js` playground files.

## How to Run

1. Open VS Code  
2. Install MongoDB extension  
3. Open any `.mongodb.js` file  
4. Click the RUN button  
5. See the output in the RESULT panel  

## Why this repo?

I created this to practice and revise all MongoDB basics in one place.
